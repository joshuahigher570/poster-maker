# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately rather than opening a public issue.

**Email:** damjan@dabotech.io

We will work with you to understand and address the issue.

## Scope

This project generates HTML files and uses Playwright to render them locally. Security concerns might include:

- Malicious content in generated HTML files
- Path traversal in file handling scripts
- Dependency vulnerabilities
